items marked with * are more advanced and may not be done in this game

Explore the underwater map of violet planet.

Game Genre: Adventure; Sci-fi

Losing condition is when the player triggers (touches) the toxic generator (saltwater attack), which is part of the planet's self-defense system. Winning condition is when the player manages to locate and turn off the planet's self-defense system.

Randomly generate new game characters, including their names, attributes and appearance. Each time the player triggers the losing condition, the unlucky character dies, and the player will restart the game with another game character. The already explored map stays explored.

*The robot that comes with every character, and player can control the robot by typing snippets in the robot control panel. This is an advanced part of game. Might not be completed. Ideas not formed yet. It aims to teach players of this game basics of programming.
